# 🧮 Simple Calculator with History

A simple command-line calculator built using **Python**. This project performs basic arithmetic operations and stores the history of all calculations performed during the program execution.

It is designed for beginners to practice Python fundamentals such as functions, loops, conditional statements, exception handling, lists, and dictionaries.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📝 Stores calculation history
- ⚠️ Handles division by zero
- ✅ Validates user input
- 📋 Menu-driven interface

## ▶️ How to Run

### Step 1: Clone the repository

```bash
git clone https://github.com/your-username/simple-calculator-with-history.git
```

### Step 2: Open the project folder

```bash
cd simple-calculator-with-history
```

### Step 3: Run the program

```bash
python calculator.py
```

> If your system uses Python 3 separately:

```bash
python3 calculator.py
```

## 📖 Functions Used

| Function | Description |
|----------|-------------|
| `add(a, b)` | Adds two numbers |
| `subtract(a, b)` | Subtracts two numbers |
| `multiply(a, b)` | Multiplies two numbers |
| `divide(a, b)` | Divides two numbers with zero division handling |
| `show_menu()` | Displays the calculator menu |
| `save_history()` | Stores calculations in history |
| `view_history()` | Displays previous calculations |
